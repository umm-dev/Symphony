# Contributing to Symphony

Thanks for contributing.

## Requirements

- Contributions are accepted only under the terms in `CLA.md`.
- By opening a PR or submitting a patch, you confirm you agree to `CLA.md`.
- Keep contributions compatible with `LICENSE` and third-party dependency
  licenses.

## AI-Assisted Contributions

AI-assisted contributions are permitted, including code or content generated
with tools such as Codex, Claude Code, AI agents, or other intelligent
automation tools.

Submitting AI-assisted material does not guarantee review, acceptance, merging,
or any specific treatment outcome. Maintainers retain full discretion to
accept, request changes, defer, or reject contributions.

If any part of a contribution is AI-assisted, include an `AI Disclosure`
section in the pull request description that states:
- Provider and tool names used.
- Files or sections materially AI-assisted.
- Confirmation that you reviewed and verified the submitted output.
- Confirmation that the provider's terms permit submitting that output in pull
  requests to this repository.

## Pull Request Checklist

- Include a clear summary of what changed and why.
- Keep changes scoped and testable.
- Confirm that you have rights to submit all included content.
- If AI assistance was used, include the required `AI Disclosure` section.
- Add this statement in the PR description:
  `I agree to the Symphony CLA in CLA.md for this contribution.`
